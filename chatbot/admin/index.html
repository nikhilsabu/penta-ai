Admin panel requires PHP hosting

GitHub Pages can only host static files. The Pentame admin dashboard uses PHP + MySQL for login, conversations, leads, and settings — so it cannot run on github.io.

You are viewing the static GitHub Pages copy. Use one of the options below for the real admin panel.
  • Local (XAMPP): http://localhost/penta-ai/chatbot/admin/
  • Production: deploy the full chatbot/ folder to PHP hosting, then open /chatbot/admin/
  • Set ADMIN_PASSWORD in chatbot/.env before logging in
